# Chat-application
Simple Chat application using following components:
- socket.io
- express.js
- node.js


Installation : 

1) Just download the project. 
2) Open the directory where project is located
3) run "node index" & and you will see command "listening on *:3000" means server is setup and ready to listen on port 3000
4) To run app, open browser and run "http://localhost:3000/"
5) To test chatting with multiple user, open the above link from another browser or tab.
